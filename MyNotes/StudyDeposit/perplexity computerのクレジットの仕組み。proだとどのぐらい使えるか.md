# perplexity computerのクレジットの仕組み。proだとどのぐらい使えるか

Perplexity Computerのクレジットは「Max以上のプラン」で使える仕組みで、現時点では個人向け**Pro**プラン単体ではクレジットはまだ付きません。ProからMaxにアップグレードすると、Computer用クレジットが使えるようになります。[^1_1][^1_2]

## クレジットの基本ルール（Computer）

- クレジットは主に「Computer」などの高負荷なプレミアム機能に使われるポイント制。通常のPerplexity Ask検索には消費されません。[^1_2][^1_1]
- タスクの複雑さに応じて消費クレジット数が変わります（コード実行・大量Web巡回・長時間タスクほど多く消費）。[^1_1][^1_2]
- 残高や使用内訳はアカウントのUsage/Credits画面で確認できます。[^1_2][^1_1]


## Proプランでどこまで使えるか

- 個人向けProプランは「クレジット制Computerなし」で、代わりに
    - 高性能モデルの利用
    - Deep Researchの「高頻度の利用枠」
    - ファイルやアプリ生成などに対する「月間の上限付き利用」
が含まれます（具体的な回数は「平均的利用を想定した月間・週間上限」という扱い）。[^1_3][^1_4]
- ただし「Computer用クレジット」は**Max/Enterprise Maxのみ付与**で、Proへのクレジット付与は「今後対応予定（on the roadmap）」とされています。[^1_1][^1_2]


## Maxにした場合のクレジット量（比較用）

- Maxユーザー: 月あたり10,000クレジット付与。[^1_2][^1_1]
- 期間限定ボーナスとして追加35,000クレジットが付くキャンペーン中（合計45,000分を先に使えるイメージ）。[^1_1][^1_2]


### プラン別ざっくりイメージ

| プラン | Computerクレジット | Deep Research/Ask 等 |
| :-- | :-- | :-- |
| Free | なし[^1_2] | Ask・Deep Researchは少数回程度[^1_3] |
| Pro | なし（今後対応予定）[^1_1][^1_2] | 高頻度で利用可（ただし内部的に上限あり）[^1_3][^1_4] |
| Max | 月10,000＋ボーナス35,000[^1_1][^1_2] | Ask無制限、Deep Researchも十分量[^1_1][^1_2] |

## 実際どのくらい使えると考えればいいか

- Pro:
    - 普通のQA・要約・論文リサーチ・コード相談などは、**日常利用ならほぼ気にせず使える**設計です（裏側でフェアユース上限はあるが公開されていません）。[^1_4][^1_3]
    - ただし、Computerのような「ブラウザエージェントで超大規模タスクを走らせる」用途は不可。
- Max:
    - Computerを使った重いタスク（例：数十サイトをクロールして長文レポート生成）を、月に数十〜数百タスク程度こなせるイメージ量のクレジットが付きます。[^1_2][^1_1]

もし「Computerで具体的に何をどれくらいやりたいか」がわかれば、Proで様子を見るか、最初からMaxにするかもう少し具体的にアドバイスできます。どんなユースケース（論文大量レビュー、コード生成、データ収集など）でComputerを使いたいですか？

# 今proプランで40000クレジットほどあるが、これは使い切りになってしまうのか。あとで回復するのか

結論から言うと、その「40000クレジット」が何の種類かで扱いが変わりますが、多くの場合は「使い切り」ではなく、毎月リセットされる“月次クレジット”か、期限付きの“ボーナス／購入クレジット”です。[^2_1]

## クレジットの種類とリセット

Perplexityには大きく3種類のクレジットがあります。[^2_1]

- **Monthly（Plan） credits**
    - サブスクリプションに含まれる月次クレジット。
    - 毎月の請求日ごとに**リセットされて満額に戻る**一方、使い残しは翌月に繰り越されません。[^2_1]
- Purchased credits（購入分）
    - 追加購入したクレジット。
    - 使うまで残りますが、「1年間まったく使用がない」と**失効**します。[^2_1]
- Bonus credits（ボーナス）
    - キャンペーンなどで付与されるもの。
    - 付与時点で示された**有効期限までに使わないと失効**します。[^2_1]

クレジットは「ボーナス → 月次（Plan） → 購入」の順に消費されます。[^2_1]

## 「40000クレジット」がどう扱われるか

- もし画面の表記が「Plan」や「Monthly」となっていれば、
    - その40000は**請求サイクルごとに毎月回復**する枠で、使い切らなくても繰り越しはされません。[^2_1]
- 「Bonus」や期限付きの表示があれば、
    - キャンペーン等の**一度きりのボーナス**で、有効期限までの“使い切り”に近い扱いです。[^2_1]
- 「Purchased」とあれば、
    - 追加購入分で、**使うまで残るが1年利用がなければ失効**します。[^2_1]


## どこで確認できるか

- 設定の「Usage / Credits」ページに、
    - Bonus / Plan / Purchased の内訳
    - 次の月次リフレッシュ日
が表示されるので、そこで自分の40000クレジットがどの種類か確認できます。[^2_1]

画面に表示されている内訳（Plan / Bonus / Purchased）のラベルがどう出ているか教えてもらえれば、「一度きりなのか・毎月回復なのか」をもっと正確に言語化できます。
